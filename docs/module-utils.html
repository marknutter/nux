(static) selector(selectorString) → {Array}

Generate a Nux reducer function given a custom reducer function.

Returns an array path that can be used to deeply select inside of Nux. 'children' strings will be interleaved between tag strings while all nodes from 'props' and onward will be added in sequence. All returned arrays will include a leading 'ui' node.
