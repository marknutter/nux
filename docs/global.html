Generate a Nux reducer function given a custom reducer function.

Create a reducer using a provided reducer combined with Nux's internal reducer. An initial vDOM UI object can be passed in to initialize the store with, as well as any options to further configure the reducer. The reducer returned is intended for use with a Redux store.
Parameters:
Name Type Attributes Description
appReducer function The provided reducer function.
initialUI Object <optional>
 The initial UI vDOM object which will become the first state to be rendered.
options Object <optional>
 Options to configure the generated reducer.
