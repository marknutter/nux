(static) renderUI(store, ui, pathArrayopt) → {Store}

Accepts a Nux vDOM object and returns a VirtualNode. The vDOM object's 'children' are recursively converted to VirtualNodes. The 'props' for any given are modified such that any custom events are assigned callbacks which dispatch the associated actions. The Nux default event handlers are also added to the 'props' object for a given node. This is where all the magic happens, folks.
